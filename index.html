Projects

1. Focus

This application helps users to practice meditation without any hassles. Users have to register to meditate. They can log in daily and meditate choosing from a variety of guided meditations available in the application. The application also motivates the users by awarding them a variety of badges. Apart from this, the 21-day challenge program which starts on the first of every month is very useful for people who want to make meditation a habit. 21-Day Challenge Page shows the progress of all the users signed up for this program.

Technologies Used:

HTML5, CSS3, JavaScript, jQuery, Node.js, Express.js, MongoDB

Live Demo:

https://focus-meditation-app.herokuapp.com/
Git URL:

https://github.com/jayshol/meditation-app.git

2. Puzzle-Mania

Puzzle-Mania is an easy to use fun game app that allows users to play Jigsaw puzzles online. The users have to register to play games. Once a user registers and logs into the app, he can choose a variety of pictures from the app's library and play the jigsaw puzzle. The app provides three levels to play the game. All the puzzles are timed, and successful attempts of the user are stored in the database.

Technologies Used:

HTML5, CSS3, JavaScript, React, Redux, Node.js, Express.js, MongoDB

Live Demo:

https://jigsaw-puzzle-app.herokuapp.com/home
Git URLs:

https://github.com/jayshol/puzzle-react.git

https://github.com/jayshol/puzzle-node-server.git

3. Restaurant Search

This app provides a list of nearby restaurants given a zip code for the search. The users can further filter their search by cuisine, vegetarian options, vegan options, by distance and by the restaurants that are open at the time of the search. The results are displayed as a list and also as markers in a Google map clicking on which gives the information about the restaurants.

Technologies Used:

HTML5, CSS3, JavaScript, jQuery, Google Maps API, Google Geocode API

Live Demo:

https://findrestaurant--jayshol.repl.co/
Git URL:

https://github.com/jayshol/restuarant-search.git
